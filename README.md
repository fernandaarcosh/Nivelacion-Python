# Nivelacion-Python
Nivelación Python MCOC
# Introducción
Nivelación guiada por videos, en donde se aplican diferentes ejercicios y tareas para poder volver a entender y aplicar lo básico del uso de Python.
# Tareas
Se realiza una tarea por dia durante 11 dias, la cual consciste en ver un video Tutorial de diferentes temas sobre como utilizar las herrramientas y códigos básicos en Python.
Estas, tienen una fecha de entrega limite en donde se debe subir el código aprendido del video, junto con un ejercicio dado por el locutor. Las tareas son las siguientes:
1. Grupo python básico:
- 1.1 Introducción.
- 1.2 If else.
- 1.3 Funciones.
- 1.4 Listas.
- 1.5 For Loops.
- 1.6 While.
- 1.7 Diccionarios.

2. Grupo numpy y scipy:
- 2.1 Numpy en 5 mins. 
- 2.2 Especial de arreglos en numpy.
- 2.3 Numpy Tutorial más completo.

3. Especial de matplotlib.

Los códigos de cada tarea se encuentran en este mismo repositorio en una carpeta con la fecha de entrega de cada uno.
